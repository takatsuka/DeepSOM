Data File

Data file that the SOM will be trained one. Supported file types: txt

Data format:

(dimension of data points) (number of data points)

(data points space separated)

Example data file:

3 2

1 2 3

0 4 1

Iterations

Maxiumum number of iterations in the SOM training.

Restrictions: Integer larger than 0. Default value: 20

More training iterations allow the SOM to model the input data more closely.

Grid shape

Shape of the SOM node grid to use for the network.

Choices: Rectangle or Hexagon. Default value: Hex

A hexagon grid is preferred as all adjacent nodes are equidistant from one another.

Test split

The proportion of the input data to be used for testing a LVQ classifier created using the SOM

Restrictions: Value between 0 and 1. Default value: 0.25

A value of 0 means a LVQ classifier will not be created at all.

Side Length

Side lengths of the grid.

Input format: 2 space separated integers. Restrictions: Integers must be larger than 0. Default value: 10 10

Larger side lengths create more nodes in the SOM and captures finer details in the input data.

Learning Rate

Initial learning rate of the SOM.

Restrictions: Value between 0 and 1. Default value: 0.5

A larger alpha adjusts how quickly the SOM initially trains and learns to model the input data.

Neighbourhood Radius

Initial neighbourhood radius of the SOM.

Restrictions: Integer larger than 0. Default value: 0

A larger neighbourhood radius allows SOM nodes to adjust their positions according to more neighbouring nodes.

A value of 0 will set sigma to be 80% of the given side length.

Decay Constant

Decay constant for learning rate and neighbourhood radius.

Restrictions: Value larger than 0. Default value: 0

Larger values lead to a slower decay in learning rate and neighbourhood radius over time.

A value of 0 will set lambda to be 80% of the given iterations.

Input Dimensions

Select the dimensions of the input data that the SOM will use.

Input format: Space separated list of indices. Restrictions: Indices are indexed starting from 1. Default value: null

Should only be used for nodes connected to the input node to allow SOM nodes to selectively learn certain dimensions.

A value of null will set the SOM to use all input data dimensions.

SOM structure drawing tool

Click the + button to add a new SOM node.

Click on SOM nodes to modify their parameters.

Shift+Click 2 SOM nodes to connect them. The output of one SOM will be fed as input into another SOM according to the arrow direction.

Ctrl+Click a SOM or connection in order to delete it.

Restrictions: The final output SOM should be connected. The input node should be connected.

The graph should have no cycles.

UMatrix Visualisation

Availability: Available for all SOMs

Provides a 2D visualisation of the distances between neurons in the SOM grid

Red nodes indicate a neuron is far from its neighbours, hence red nodes represent cluster boundaries.

Blue nodes indicate a neuron that is close to its neighbours, hence areas of blue nodes represent a cluster.
