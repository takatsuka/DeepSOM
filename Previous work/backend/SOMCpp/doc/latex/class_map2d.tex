\doxysubsection{Detailed Description}
Implementation of \mbox{\hyperlink{class_s_o_m}{S\+OM}} in 2D. 

Basic square 2D \mbox{\hyperlink{class_s_o_m}{S\+OM}} with a given side length. \mbox{\hyperlink{class_s_o_m}{S\+OM}} training parameters are specified in initialisation (constructor).
