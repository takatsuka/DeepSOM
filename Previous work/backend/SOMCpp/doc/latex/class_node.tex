\doxysubsection{Detailed Description}
Class for \mbox{\hyperlink{class_s_o_m}{S\+OM}} nodes. 

\mbox{\hyperlink{class_s_o_m}{S\+OM}} node class supporting topological coordinates and feature space coordinates. The node\textquotesingle{}s neighbors are decided based on the topology of the \mbox{\hyperlink{class_s_o_m}{S\+OM}}. The feature space coordinate matches the dimensionality of the input vectors.
